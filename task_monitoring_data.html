In SaaS Backup for Microsoft Office 365, you can monitor user data for all services like email or url addresses, mailbox types, license use, discovery state, last discovery, backup status, backup tier, and more.

Steps

  1. Click Services.

  2. Click the service for which you want to export user data.

    1. For Microsoft Exchange Online, click on the number of protected, pending, or unprotected mailboxes.

    2. For Microsoft SharePoint Online, click on the number of protected, pending, or unprotected sites.

    3. For Microsoft OneDrive for Business, click on the number of protected, pending, or unprotected MySites.

    4. For Microsoft Office 365 Groups, click on the number of protected, pending, or unprotected groups.

  3. Click Download to export an Excel file of user data for the respective service.
